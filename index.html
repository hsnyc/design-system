Design System

A framework-agnostic design system for building consistent user interfaces. This design system provides reusable components and design guidelines that can be used in any project regardless of the framework.

Note: Browse components and guidelines using the navigation menu.

Getting Started

This design system is built with plain HTML, CSS, and JavaScript, making it compatible with any web project. No build steps or frameworks required.

Installation

To use this design system in your project:

  1. Download the design system files
  2. Include the necessary CSS and JavaScript files
  3. Copy the HTML markup for the components you need
<!-- Include the core styles -->
<link rel="stylesheet" href="styles/variables.css">
<link rel="stylesheet" href="styles/global.css">

<!-- Include component styles as needed -->
<link rel="stylesheet" href="components/button/button.css">
<link rel="stylesheet" href="components/card/card.css">

<!-- Include theme -->
<link id="theme-stylesheet" rel="stylesheet" href="styles/light-theme.css">

<!-- Include scripts -->
<script src="scripts/main.js"></script>

Colors

The color system is built using CSS variables, making it easy to customize and extend.

Color Palette

Primary
var(--color-primary)
Secondary
var(--color-secondary)
Accent
var(--color-accent)
Neutral
var(--color-neutral)

Usage

Use CSS variables to apply colors consistently throughout your application:

.my-element {
  background-color: var(--color-primary);
  color: var(--color-base-100);
}

.my-accent-element {
  background-color: var(--color-accent);
  border-color: var(--color-accent-content);
}

  --font-size-lg: 1.125rem;
  --font-size-xl: 1.25rem;
  --font-size-2xl: 1.5rem;
  --font-size-3xl: 1.875rem;
  --line-height-base: 1.5;
  --line-height-heading: 1.2;
}

Navigation Component

The Navigation component provides a responsive navigation bar for your application.

Example

The navigation bar at the top of this page is an example of the Navigation component.

Features include:

  • Responsive design that adapts to different screen sizes
  • Mobile menu for small screens
  • Dropdown menus for organizing navigation items
  • Theme toggle button for switching between light and dark modes

Usage

To use the Navigation component in your project:

<!-- Include the navigation CSS -->
<link rel="stylesheet" href="components/navigation/navigation.css">

<!-- Include the navigation component -->
<div id="navigation-container"></div>

<!-- Load the navigation -->
<script>
  fetch('components/navigation/navigation.html')
    .then(response => response.text())
    .then(html => {
      document.getElementById('navigation-container').innerHTML = html;
      
      // Initialize theme toggle
      const themeToggle = document.getElementById('theme-toggle');
      const themeStylesheet = document.getElementById('theme-stylesheet');
      
      if (themeToggle && themeStylesheet) {
        themeToggle.addEventListener('click', () => {
          if (themeStylesheet.getAttribute('href') === 'styles/light-theme.css') {
            themeStylesheet.setAttribute('href', 'styles/dark-theme.css');
            document.querySelector('.sun-icon').style.display = 'block';
            document.querySelector('.moon-icon').style.display = 'none';
          } else {
            themeStylesheet.setAttribute('href', 'styles/light-theme.css');
            document.querySelector('.sun-icon').style.display = 'none';
            document.querySelector('.moon-icon').style.display = 'block';
          }
        });
      }
    });
</script>
