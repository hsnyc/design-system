Alert Component

Alerts provide contextual feedback messages for user actions.

Example

Success! Your operation was completed successfully.
Info! This is an informational message.
Warning! This action might have consequences.
Danger! An error has occurred.

Documentation

Alerts should be used to provide feedback to users about their actions or system status.

<div class="alert alert-success">
    <strong>Success!</strong> Your operation was completed successfully.
</div>
